Canvas GPA Calculator

Installation Guide

Created by Shawn Georgie

Part 1: Download the Extension

1 Download from GitHub

Go to: https://github.com/shawngeorgie06/canvas-gpa-calculator

Click the green "Code" button, then click "Download ZIP"

2 Extract the ZIP file

Find the downloaded file (usually in your Downloads folder)

Right-click and select "Extract All"

Remember where you extract it - you'll need this location later

Part 2: Install in Chrome

3 Open Chrome Extensions

Open Google Chrome

Type chrome://extensions in the address bar and press Enter

4 Enable Developer Mode

In the top-right corner, toggle ON "Developer mode"

5 Load the Extension

Click "Load unpacked" button (top-left)

Navigate to the extracted folder: canvas-gpa-calculator-master

Click "Select Folder"

Success! You should now see "Canvas Grade & GPA Calculator" in your extensions list with a puzzle piece icon in your Chrome toolbar.

Part 3: Connect to Canvas

6 Get Your Canvas API Token

Log in to your school's Canvas website

Click your profile icon (top-right corner)

Click "Settings"

Scroll down to "Approved Integrations"

Click "+ New Access Token"

Enter a purpose (e.g., "GPA Calculator")

Click "Generate Token"

IMPORTANT: Copy the token immediately - you won't be able to see it again!

Keep your token secure! It grants access to your Canvas account. Never share it with anyone.
7 Configure the Extension

Click the extension icon (puzzle piece) in Chrome toolbar

Go to the "Settings" tab

Enter your school's Canvas URL:

  • NJIT: https://njit.instructure.com
  • Rutgers: https://rutgers.instructure.com
  • Other schools: https://[your-school].instructure.com

Paste your API token

Click "Save & Connect"

8 Select Your Grading Scale

In Settings, find "Grading Scale"

Select the scale that matches your school:

  • NJIT: A=4.0, B+=3.5, B=3.0, C+=2.5, C=2.0, D=1.0
  • Standard Plus/Minus: A=4.0, A-=3.7, B+=3.3, B=3.0, etc.
  • Standard: A=4.0, B=3.0, C=2.0, D=1.0
  • Custom: Define your own values

Part 4: Using the Extension

Dashboard Tab

  • Semester GPA: Your GPA for the selected semester
  • Cumulative GPA: Average of all completed semester GPAs
  • Quick Course Impact: Shows which courses to focus on for max GPA improvement
  • Target GPA Planner: Calculate what GPA you need to reach your goal

Courses Tab

  • View all courses for the selected semester
  • Edit grades, credits, or semester for any course
  • Add manual courses (for classes not in Canvas)
  • Exclude courses from GPA calculation

Features

  • Auto-Sync: Grades update automatically every 15 minutes
  • Dark Mode: Click the moon icon to toggle
  • Upcoming Semesters: Automatically excluded from cumulative GPA
  • N/A Grades: Use for courses without grades yet

Troubleshooting

"Invalid API token" error?

Make sure you copied the entire token. Generate a new one if needed.

Courses not showing?

Check that your Canvas URL is correct (include https://)

Make sure you're enrolled in active courses

Wrong GPA calculation?

Verify you selected the correct grading scale for your school

Check that course credits are correct (click to edit)
